# 🎮 Premium Mechanical Keyboard Showcase

A high-performance, interactive product showcase built with Next.js 14, Framer Motion, and HTML5 Canvas.

## ✨ Key Features

- **🚀 Scroll-Linked 3D Scrubbing**: Perfectly synchronized product disassembly/assembly animation using a high-density image sequence.
- **🎨 HTML5 Canvas Performance**: Leveraging bitmapped frame rendering for 60fps responsiveness.
- **✨ Parallax Overlays**: Dynamic storytelling text that slides and fades based on user interaction.
- **🛡️ Glassmorphism Design**: Minimalist, premium UI components with ambient glows and backdrop blurs.
- **📱 Responsive Layout**: Fully optimized for desktop and mobile viewing with `object-fit: cover` logic.

## 🛠️ Built With

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion**
- **Lucide React** (Icons)

## 🏁 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/XDX-DREADX/Keyboard-Showcase.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) to view the showcase.
